# **App Name**: RashExam Analyzer

## Core Features:

- OMR Scanning: Scan test answer sheets using the mobile device camera and extract answers. Supports automatic processing of up to 100 tests for newly registered teachers.
- RASH Module Assessment: Assess tests based on the RASH module, generating scores and reports for each student and determine grade level(C, C+, B, B+, A, A+).
- Report Generation: Generate detailed reports, charts, and tables based on test results.
- Data Import/Export: Import student data from Excel and export results to Excel.
- User Authentication: Secure login system using login and password. Implement Telegram bot verification for login and password recovery, ensuring one user per bot.
- OMR Template Customization: Allow admin to configure OMR answer sheet templates, adjusting for question numbers and uploading the school logo. Templates should be A4 format with bubble-style answer markings, similar to ZipGrade.com.
- Telegram Bot Integration: Integrate a Telegram bot to send results to groups specified by the teacher and allow students to retrieve their last result by entering their ID.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to evoke trust and professionalism.
- Background color: Light gray (#F0F0F0), a desaturated version of the primary color, to ensure readability and reduce eye strain.
- Accent color: Soft green (#77DD77), analogous to blue, for positive feedback and calls to action.
- Body and headline font: 'Inter', a sans-serif font known for its clarity and readability in digital interfaces, is suitable for both headings and body text.
- Code font: 'Source Code Pro' for displaying code snippets related to the grading system or module configurations.
- Use clear, intuitive icons for navigation and actions, ensuring they are easily understandable and visually consistent.
- Design a responsive layout optimized for mobile devices, with a simple and clear interface for easy navigation and data input.
- Subtle animations to provide feedback and enhance user experience during actions such as scanning and report generation.
